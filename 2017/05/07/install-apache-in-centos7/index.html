1. 安装centos系统

首先安装centos系统，这个比较简单，只是注意安装的时候不要最小化安装，否则系统内容比较少，需要自己重新配置。新手的话建议安装带有图形界面的centos系统，方便自己检查和验证，KDE界面做的还是不错的。

3. 测试

访问你的服务器地址，如果显示Apache的测试页面，说明httpd服务已经成功启动了。
